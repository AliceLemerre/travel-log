# Nom du projet
Travel log

## Description
Application de carnet de voyage pour enregistrer toutes ses étapes de vacances, avec dates, notes et photos inclus.

## Fonctionnalités
- [ ] Authentification
- [ ] CRUD [voyages]
- [ ] CRUD [étapes]
- [ ] CRU [user]
- [ ] Recherche par label, par région, par pays, par ville, par fourchette de dates
- [ ] Filtres asc/desc

## Stack technique
- React 18
- Vite
- Supabase
- CSS natif

## Installation
git clone https://github.com/AliceLemerre/travel-log.git
cd travel-log
npm install

### Prérequis
- Node.js 18+
- npm ou yarn

### Lancement en local
npm run dev
